## 🌦️ Weather CLI

### Objective

>- A simple Command Line Interface (CLI) application to fetch weather data using Python.
>- This project demonstrates the use of API calls, modular coding, testing with pytest, and CI/CD with GitHub Actions.


### Tech Stacks
>- Python
>- YML
>- git
>- GitHub
>- Pytest
>- Weather API
>- CMD
>- Docker
